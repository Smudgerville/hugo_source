Getting Started with Hugo & R

Overview The aim of this post is to describe how to get up and running with a static site using Blogdown and Github Pages. All of these tools make it fantastically easy to build a quality website quickly and easily. We will use two git repositories, one to build the site and another to publish in a format that allows the site to be statically hosted with github pages. Understanding the Tools Blogdown: An open-source (GPL-3) R package to generate static websites based on R Markdown and Hugo.
